
# Email Automation System

## Description
This project automatically sends emails using Python and SMTP.

## Features
- Automatic email sending
- Gmail SMTP integration
- Beginner-friendly automation project

## Technologies Used
- Python
- SMTP

## How to Run

1. Install Python
2. Open project in VS Code
3. Add your Gmail and App Password in main.py
4. Run the project:

```bash
python main.py
```
